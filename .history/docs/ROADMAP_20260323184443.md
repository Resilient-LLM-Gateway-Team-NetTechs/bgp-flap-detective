# Build Roadmap

## Phase 1: Foundation (completed)

- Workspace scaffolding
- MCP server prototype with core tools
- Unit tests for parser logic
- VS Code tasks for run and test

## Phase 2: Reliability

- Add vendor-specific parser modules and fixtures
- Add connection retry/backoff and timeout tuning
- Add richer exception handling and user-facing diagnostics
- Add test matrix for representative CLI outputs

## Phase 3: Usability

- Add inventory validation command
- Add aggregate health tool across all devices
- Add root-cause confidence scoring
- Add markdown report export option

## Phase 4: Production Readiness

- Replace in-file secrets with secure secret source
- Add structured logging and audit metadata
- Add CI for lint, tests, and packaging checks
- Add release tagging and changelog process
